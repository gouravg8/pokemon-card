## Setup

1. Prerequisite(pnpm)
   ```bash
   npm i -g pnpm
   ```

2. Clone the repository:

   ```bash
   git clone https://github.com/gouravg8/pokemon-card.git
   ```

3. Navigate to the project directory:

   ```bash
   cd pokemon-card
   ```

4. Install the dependencies:

   ```bash
   pnpm install
   ```
5. Start the development server:
   ```bash
   pnpm run dev
   ```
